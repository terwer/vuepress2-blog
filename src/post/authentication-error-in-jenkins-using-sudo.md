# 解决jenkins使用sodo出现的Authentication error in jenkins on using sudo错误

修改suduousers，

```bash
sudo su    
visudo -f /etc/sudoers
```

运行jenkins无密码使用sudo

```bash
jenkins ALL= NOPASSWD: ALL
```
