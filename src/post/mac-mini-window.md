# Mac最小化窗口的方法

使用快捷键 `Command+Option+H`，可以实现快速**隐藏**除当前应用程序之外所有程序窗口的目的。 使用快捷键 `Command+Option+M+H`，可以实现快速**隐藏**所有应用程序窗口的目的。
