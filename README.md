# vuepress2-blog

My blog auto build using vuepress2 and vercel

[preview](https://www.terwer.space)

## Install

```bash
pnpm install
```

## Dev

```bash
pnpm dev
```

## Update

```bash
pnpm update
```

Output directory is `src/.vuepress/dist`
