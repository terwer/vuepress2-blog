# vuepress2-blog

My blog auto build using vuepress 2 and vercel

[preview](https://terwer.space)

## Install

```bash
pnpm install
```

## Dev

```bash
pnpm dev
```

## Update

```bash
pnpm update
```

Output directory is `src/.vuepress/dist`
